What problem does this project address?
This Project addresses the need for a local public transport app.
What is the goal of this project?
The goal of this project is to produce a working html/javascript web app that will allow a user to call for buss stop information from the web app, recieving its time to next bus.
What are some key features that define the product?
  • Uses web technologies: HTML and CSS and javascript(Express and nodejs).

What makes this product different from others?
The project does not try to reinvent the wheel rather then to define a new wheel with better specifications.
What are we not going to do?
This project does not attempt to reinvent the wheel on transport apps instead atempts to simplfy the process of a transport app.
